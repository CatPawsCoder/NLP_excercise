# "AutoSummarizer" 
Проект для составления конспектов лекций. 

Берлов Илья
Наш проект направлен на автоматизированное создание конспектов лекций и выделение основных тематических блоков в учебных материалах. Для этого мы используем различные техники обработки естественного языка (Natural Language Processing, NLP) и методы машинного обучения. Проект предоставляет инструмент для автоматизированного анализа и конспектирования лекций, а также для извлечения ключевых тем из текста с использованием методов машинного обучения из библиотеки scikit-learn. Проект охватывает несколько ключевых этапов:
Чтение лекций: Производится чтение содержания лекций из текстовых файлов.
read_lecture(file_path): Загружает содержание лекции из файла по указанному пути. Содержание сохраняется в глобальной переменной lecture_content.
Обработка текста лекций: Лекции подвергаются предварительной обработке, включая очистку от лишних символов и токенизацию. Учитывается частота слов для определения их важности.
process_lecture(): Очищает лекцию от лишних символов, токенизирует текст, удаляет стоп-слова и вычисляет частоту слов. Затем определяет роль каждого предложения в лекции на основе частоты слов. Использует TF-IDF (Term Frequency-Inverse Document Frequency) для выделения ключевых предложений, основываясь на их роли и косинусовой схожести. Возвращает итоговое резюме лекции.
Извлечение тем с использованием LDA: Применяется Latent Dirichlet Allocation (LDA) для выделения тем в содержании лекций
extract_topics(num_topics=5): Извлекает темы из лекции, используя метод Latent Dirichlet Allocation (LDA). Удаляет стоп-слова, токенизирует текст и применяет CountVectorizer для подготовки данных. Затем применяет LDA для извлечения заданного количества тем. Возвращает веса тем и топ-10 слов для каждой темы.
Основные библиотеки:
re: Используется для регулярных выражений при очистке лекции.
nltk: Используется для токенизации, удаления стоп-слов и анализа предложений.
sklearn: Используется для векторизации текста, вычисления косинусовой схожести и применения LDA.
Проход по лекциям: Проект обрабатывает несколько лекций, представленных в списках файлов lecture_files. Для каждой лекции выводится её резюме и темы.
Оценка качества: Для оценки качества автоматически сгенерированных конспектов используется метрика косинусного сходства между автоматическим и эталонным резюме.
